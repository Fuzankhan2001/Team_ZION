# LifeLink: Hospital Bed & Ambulance Availability Tracker

**Problem Statement ID:** HC005
**Domain:** Healthcare

## Overview
LifeLink is an intelligent, real-time command center designed to track ICU/general beds, ventilators, and ambulances across a city's hospital network. It replaces reactive phone coordination with instant, verified asset allocation.

*(Note: Documentation is actively being updated during the 24-hour hackathon development phase.)*

## Tech Stack
* **Frontend:** React.js
* **Backend:** FastAPI (Python)
* **Database:** PostgreSQL
* **Hardware/IoT:** Python Telemetry Simulators (MQTT)
* **AI/Logic:** Deterministic NLP Triggers
